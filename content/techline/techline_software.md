# Techline software

The Techline terminal is a computer, also referred to as terminal, which runs on Microsoft Windows and has Tech2 software installed. There are various versions of Tech 2 software to work with.

## SAAB
* TIS2000
* GlobalTIS 2010 / 146
* GlobalTIS 2011 / 148

## Opel / Vauxhall
* GlobalTIS v25.0
* GlobalTIS v32.0
* GlobalTIS v36.0
* GlobalTIS v40.0
